# Phase 1: Basics & Data Ingestion
This folder contains my initial work on pulling financial data and analyzing market risk.

### Key Learnings:
* Using `yfinance` to fetch historical stock data for US (AAPL) and Indian (RELIANCE) markets.
* Calculating **Daily Returns** to normalize price changes.
* Visualizing risk distributions using **Histograms** and **KDE** (Kernel Density Estimation).I learnt about Volatility and The Outlier and Centre bias.
